## Introduction

Convergence UI is a modern component library built with React, TypeScript, and Tailwind CSS. It features a unique glassmorphism design system with animated backgrounds, smooth transitions, and full dark mode support.

<br />

## Getting Started

Install the package:

<Code code="pnpm add @convergence/ui">
  <CodeHeader copyButton>Installation</CodeHeader>
  <CodeBlock lang="bash" writing cursor />
</Code>

Include the providers in your `App.tsx`:

<Code code={`import { ThemeProvider } from "@convergence/ui/providers";

function App() {
  return (
    <ThemeProvider defaultTheme="light">
      <YourApp />
    </ThemeProvider>
  );
}`}>
  <CodeHeader copyButton>App.tsx</CodeHeader>
  <CodeBlock lang="tsx" writing cursor />
</Code>

<br />

## Features

- **Glassmorphism Design** - Modern glass-effect components with backdrop blur and translucency
- **Animated Backgrounds** - Floating orb animations with customizable colors
- **Dark Mode** - Full light and dark theme support with smooth transitions
- **Accessible** - Built with accessibility in mind using Radix UI primitives
- **TypeScript** - Full TypeScript support with exported types
- **Tailwind CSS v4** - Built on the latest Tailwind with CSS custom properties

## Design Tokens

The library exposes CSS custom properties for easy customization:

<Code code={`:root {
  /* Background gradients */
  --bg-gradient-start: #f0f4ff;
  --bg-gradient-mid: #e8eeff;
  --bg-gradient-end: #f2f6ff;

  /* Animated orb colors */
  --orb-color-1: #c7d2fe;
  --orb-color-2: #ddd6fe;
  --orb-color-3: #bfdbfe;
  --orb-opacity: 0.3;

  /* Glassmorphism */
  --glass: hsla(0, 0%, 100%, 0.7);
  --glass-border: hsla(240, 6%, 90%, 0.5);
}`}>
