The Button component is the primary interactive element for user actions.

## Usage

```tsx
import { Button } from "@convergence/ui";

<Button primary label="Click me" onClick={() => {}} />;
```

## Variants

### Primary Button

Use for primary actions like form submissions.

## Theming

The Button uses CSS custom properties for theming:

| Variable          | Description      |
| ----------------- | ---------------- |
| `--button-bg`     | Background color |
| `--button-text`   | Text color       |
| `--button-border` | Border color     |

## Accessibility

- Always provide a descriptive `label`
- Use `aria-label` for icon-only buttons
- Ensure sufficient color contrast
