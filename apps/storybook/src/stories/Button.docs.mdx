The Button component is the primary interactive element for user actions.

## Usage

```tsx
import { Button } from "@convergence/ui/components";

<Button variant="default">Click me</Button>
<Button variant="destructive">Delete</Button>
<Button variant="outline" size="sm">Small Outline</Button>
```

## Accessibility

- Use clear, descriptive text for button content
- Use `aria-label` for icon-only buttons
- Ensure sufficient color contrast between variants
- The component includes proper focus states for keyboard navigation
